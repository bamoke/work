The ssh-keygen(1) manual talks about creating an OpenSSH cert authority and signing certificates, should you be interested. (Note: this doesn't use X.509, only OpenSSH's own cert format.) Usually the certificates are only useful if you have a massive amount of users (and/or servers) but don't want to use Kerberos.
